# l10n-moldova

Localizare Odoo 19 pentru Republica Moldova.

## Surse oficiale

- **Planul general de conturi contabile** — Ordinul Ministerului Finanțelor nr. 119 din 06.08.2013,
  https://mf.gov.md/sites/default/files/legislatie/Planul%20general%20de%20conturi%20contabile.pdf
  (conturi de gradul I, clasele 1-9)
- **Standardele Naționale de Contabilitate (SNC)** — Ordinul MF nr. 118 din 06.08.2013
- **Codul Fiscal al Republicii Moldova** — cote TVA (20%/8%/0%), impozit pe profit (12%),
  asigurări sociale (24%), asigurare medicală (9%)

## Module

- `l10n_md` — plan de conturi, taxe, bănci, poziții fiscale, termene de plată

## Status

Modul de bază funcțional pe Odoo 19 (instalare + teste OK). Conturile de gradul II (subconturi
pe 4 cifre), rapoartele fiscale (D300 etc.) și celelalte module (bancă, salarizare, e-factura)
nu sunt încă implementate.
